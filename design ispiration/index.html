For my site design, I took a lot of inspiration from static and web design precedent (as collated on this Pinterest board of mine). I tried to emulate some almost exactly for practice in creating certain visual modes in my own coding – but due to Pinterest's nature, it can be hard to find the credit.

If I could start again, I would be more wary of crediting where my visual output/organisation was inspired or taken from. I would also never do such 1:1 copies outside an educational context.

This board has many different examples of visual precedent – anti-design, brutalism, minimalism, web-aesthetic, old-internet aesthetic and creative data visualisation. I hope these influences are visible in the design I undertook throughout this workbook.
