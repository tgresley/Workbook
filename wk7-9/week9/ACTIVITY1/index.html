SERIAL_IN POTENTIOMETER COLOUR

In workshop we learned how to combine p5 and Arduino code and processes through serial connection to make both technologies interact. In this case, the (dataIn) value was used to change the opacity/tonal quality of a p5 canvas based on the physical potentiometer on the Arduino circuit.

Combining the different codes was much easier than expected. It was key to include the relevant p5 libraries and serial connections to ensure everything worked smoothly to create the final visual output.
