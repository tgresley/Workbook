HOME EXPERIMENTS WITH DISPLAYS

With the endless possibility of code and open-source knowledge, I spent quite a lot of time on Tinkercad finding cool LCD and digit segment projects. In these projects I was able to experiment with screen contrast through potentiometers, run stories with timed phrases, use buttons to create interactive games, and implement everyday functions such as counting/timing. These experiments deepened my appreciation of the digital world around us.
