ULTRASONICSENSOR + BUZZER CIRCUIT
Moving into arduino.ide, with a new circuit of ultrasonic sensor and active buzzer, the code language was unfamiliar and quite confusing to me. Especially after learning p5*js, another javascript language has become hard for me to grasp. What helped though is that c++ (which arduino.ide is written in) has a similar logic structure of setup and then repeated loop of code to p5; 

    void setup() {
    // runs once
    }
      
    void loop() {
    // runs forever
    }
At first, the code didn't work once uploaded, but once double checking any pin connections in the code and in the circuit, and uploading to the specific board, it began to work. This activity made me realise how carefullly you need to check if the code and board match up, and whether previous code isnt uplaoded unto the board. In arduino.ide, second nature is to compile and then uplaod immediately.

After this, some groups began changing distance values within the code to adjust how sensitive the sensor was, but we ran out of time.
