CustomFonts

We also learnt how to set custom fonts in our html pages with any ttf file

on this page, for example, i was able to download and set the original Microsoft Sans Serif from Windows 97

i did this by setting a new font family in CSS and attaching the ttf as a file - heres the code:

            @font-face {
            font-family:MSsansserif;
            src: url(../wk3/ms-sans-serif-1.ttf);
            }
